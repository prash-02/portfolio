# ⚡ Prashant Kushwaha — ECE Portfolio

A modern, interactive 3D portfolio website showcasing my work as an Electronics & Communication Engineering student. Built from scratch with **React + Vite**, **Tailwind CSS**, and **Three.js** for immersive 3D graphics.

**Live:** https://prash-02.github.io/portfolio/

---

## 🌟 Features

- **Interactive 3D Scene** — Real-time WebGL rendering with animated orb, neon rings, and particle effects
- **Smooth Animations** — Scroll-triggered reveals, parallax effects, and glassmorphism UI
- **Full-Stack Showcase** — Projects, skills, experience, achievements, and contact sections
- **Responsive Design** — Mobile-friendly with adaptive layouts
- **Dark Futuristic Theme** — Custom cyberpunk aesthetic with cyan, amber, and pink accents

## 🚀 Local Development

### Prerequisites
- Node.js 18+ and npm

### Steps

```bash
# 1. Clone or unzip this project
cd portfolio

# 2. Install dependencies
npm install

# 3. Start the dev server
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

---

## 🚀 Deployment

This portfolio is deployed to **GitHub Pages** and automatically updates on every push to the `gh-pages` branch. The build and deployment are handled via:

```bash
npm run deploy
```

This command builds the project and pushes the optimized output to GitHub Pages.
